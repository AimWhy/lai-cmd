# lai-cmd

這是一個用於生成網頁開發環境的 CLI，他基於 npx。

[English](https://github.com/LaiJunBin/lai-cmd/tree/develop#lai-cmd)｜繁體中文

---

## 🚨 **專案維護更新說明** 🚨
> 因應 ESLint 重大升級至 9.x 版本，以及目前前端初始化工具生態日益成熟（不再需要繁重的環境設定），因此決定暫時停止維護此專案。

## 環境要求
* Node.js >= 18.0.0

## 使用方式:

```
$ npx lai-cmd@latest init
```
